conditional sentences

সহজ ভাষায় If clause যুক্ত বাক্যকে conditional sentence বলে।

Conditional Sentence মুলত ৪ প্রকার ।

  1.   zero conditional sentences
  2.   First Conditional sentences.
  3.   Second Conditional sentences
  4.   Third Conditional sentences
1) Zero Condition :
If যুক্ত clause কি যদি present tense হয় এবং চিরন্তন সত্য, বৈজ্ঞানিক সত্য ও অভ্যাসগত সত্য অর্থ প্রকাশ করে তাহলে পরবর্তী clause টি present indefinite tense হয়।

Structure :

" If + Present 1st + Present 1st "

Example :

  1.   If the iron heats it turns into red.
  2.   If it rains, my car gets wet.
  3.   If you don’t brush your teeth, you get cavities.
  4.   If you heat water to 100°, it boils.
2) First Condition :
শর্ত সাপেক্ষে যে কাজটি ভবিষ্যতে হওয়ার সম্ভাবনা রয়েছে তা প্রকাশ করতে first conditional sentence ব্যবহৃত হয়।

Structure :

" If + Present 1st + will + v1 "

Example :

  1.   If you come, I will go.
  2.   If you rest , you will feel better.
  3.   If they’re hungry, I’ll make some sandwiches.
  4.   My sister will cry if she watches this movie.
3) Secound Condition :
সাধারণত ভবিষ্যতের কোন ঘটনা বোঝাতে ব্যবহৃত হয় যা সত্য হবার কোন সম্ভাবনা নেই। “হতে পারতো কিন্তু আসলে হয়নি” অর্থ প্রদান করে।

( বর্তমানের কোন ঘটনা যা অসম্ভব বা সত্য নয় তা বোঝাতেও ব্যবহৃত হতে পারে। )

Structure :

" If + past 1st + sub + would / could + v1 "

Example :

  1.   If i came he would go.
  2.   I would answer if he called.
  3.   If I were rich, I’d never work again.
  4.   If I studied, I would pass.
4) Third Condition :
সাধারণত অতীত সম্পর্কে কথা বলে এবং এমন একটি ঘটনা বর্ণনা করে যা ঘটেনি এবং ঘটনাটির ফলাফল কল্পনা করে।

Structure :

" If + past perfect + would have / could have + v3 "

Example :

  1.   If I had gone, he would have came.
  2.   If we had left earlier, we would have arrived on time.
  3.   If Bonnie had studied, she would have passed the test.
  4.   If I hadn’t learnt English, I wouldn’t have got this job.
